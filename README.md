## MAC

```bash
sudo -v
curl -sSfL https://raw.githubusercontent.com/linzeyan/terminal_config/master/macos/init | bash
```

or

```bash
wget -qO- https://raw.githubusercontent.com/linzeyan/terminal_config/master/macos/init | bash
```
